# <h1> 2D-Souls </h1>
⚔️ This is a custom fan made game of the popular franchise Dark Souls!
<br><br>
Dark Souls is RPG series that is famous for its difficulty. Because the game and its idea really speaks to me I made a 2D version of it. Credits to From Software and there fantastic team for making the Blood Souls games (Dark Souls & Bloodborne). Sekiro ofcourse too 🙏🏻.
